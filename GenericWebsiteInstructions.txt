## IMAGE GENERATION

Use the Google Gemini API with the **gemini-3-pro-image-preview** model which supports 2K and 1K output sizes.

### Installation Requirements
```bash
# For cwebp (libwebp) - WebP conversion
sudo apt-get install webp

# For Sharp (Node.js alternative)
npm install sharp

# Google Generative AI SDK
pip install google-generativeai --break-system-packages
# or for Node.js
npm install @google/generative-ai
```

### Image Generation Workflow

**Step 1: Generate Image with Gemini 3 Pro**
- Use model: `gemini-3-pro-image-preview`
- Set output size: 2K for hero images (16:9), 1K for square images (1:1)
- Gemini outputs PNG format

**Python Example:**
```python
import google.generativeai as genai
from google.generativeai import types
import base64
import os

genai.configure(api_key=os.environ["GEMINI_API_KEY"])

def generate_image(prompt, output_path, aspect_ratio="16:9"):
    """
    Generate image using Gemini 3 Pro
    aspect_ratio: "16:9" for hero images (2K), "1:1" for square images (1K)
    """
    model = genai.GenerativeModel("gemini-3-pro-image-preview")
    
    # Use 2K for hero (16:9), 1K for square (1:1)
    image_size = "2k" if aspect_ratio == "16:9" else "1k"
    
    response = model.generate_content(
        contents=prompt,
        generation_config=types.GenerationConfig(
            response_modalities=["image"],
            image_generation_config=types.ImageGenerationConfig(
                output_mime_type="image/png",
                aspect_ratio=aspect_ratio,
                output_compression_quality=100,
                output_image_size=image_size
            )
        )
    )
    
    # Save the image
    for part in response.candidates[0].content.parts:
        if part.inline_data:
            image_data = base64.b64decode(part.inline_data.data)
            with open(output_path, "wb") as f:
                f.write(image_data)
            return output_path
    
    return None

# Usage:
generate_image(
    "Professional house cleaning service, modern home interior, bright and clean",
    "temp_hero.png",
    aspect_ratio="16:9"
)
```

**Step 2: Convert PNG to WebP**
- Use `cwebp` (from libwebp) or `Sharp` (Node.js) to convert PNG to WebP
- Example using cwebp: `cwebp -q 85 input.png -o output.webp`
- Example using Sharp:
  ```javascript
  const sharp = require('sharp');
  await sharp('input.png').webp({ quality: 85 }).toFile('output.webp');
  ```

**Step 3: Clean Up**
- Delete the original PNG file after successful conversion
- Only keep the final WebP file

**Step 4: Save to /images/ Folder**
- Store all final WebP images in the `/images/` folder
- Use descriptive filenames: `hero-[service-name].webp`, `about-us.webp`, `service-[name].webp`

### Complete Example Script

```python
import google.generativeai as genai
from google.generativeai import types
import subprocess
import base64
import os

genai.configure(api_key=os.environ["GEMINI_API_KEY"])

def generate_and_process_image(prompt, output_name, aspect_ratio="1:1", quality=85):
    """
    1. Generate image with Gemini 3 Pro (2K for hero, 1K for square)
    2. Convert to WebP
    3. Clean up temp files
    """
    temp_png = f"temp_{output_name}.png"
    final_webp = f"images/{output_name}.webp"
    
    # Ensure images directory exists
    os.makedirs("images", exist_ok=True)
    
    # Step 1: Generate with Gemini 3 Pro
    # Use 2K for hero (16:9), 1K for square (1:1)
    image_size = "2k" if aspect_ratio == "16:9" else "1k"
    
    model = genai.GenerativeModel("gemini-3-pro-image-preview")
    
    response = model.generate_content(
        contents=prompt,
        generation_config=types.GenerationConfig(
            response_modalities=["image"],
            image_generation_config=types.ImageGenerationConfig(
                output_mime_type="image/png",
                aspect_ratio=aspect_ratio,
                output_compression_quality=100,
                output_image_size=image_size
            )
        )
    )
    
    for part in response.candidates[0].content.parts:
        if part.inline_data:
            image_data = base64.b64decode(part.inline_data.data)
            with open(temp_png, "wb") as f:
                f.write(image_data)
    
    # Step 2: Convert to WebP with compression
    subprocess.run([
        'cwebp',
        '-q', str(quality),
        temp_png,
        '-o', final_webp
    ])
    
    # Step 3: Clean up
    os.remove(temp_png)
    
    return final_webp

# Usage:
# Hero image (16:9 aspect ratio)
generate_and_process_image(
    "Professional house cleaning service, modern home interior, bright and clean, photorealistic",
    "hero-house-cleaning",
    aspect_ratio="16:9",
    quality=85
)

# Card image (1:1 square aspect ratio)
generate_and_process_image(
    "Deep cleaning service, detailed cleaning supplies, professional",
    "service-deep-cleaning",
    aspect_ratio="1:1",
    quality=85
)
```
